Step 1: Go to console.firebase.google.com
Open your project ketan-finance
Click Firestore Database in the left menu
Click Create database → choose Production mode → select any location → Done
Then: Click Rules tab in Firestore and paste this rule:

rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /{doc=**} {
      allow read, write: if true;
    }
  }
}
Click Publish
